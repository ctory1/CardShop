# CardShop

## The Goal:
- Create software for our cardshop business that advertises cards visually on a screen.
- Create a website to advertise what we do.

## Refresh local code

When frontend JavaScript/CSS/HTML or API code changes, run this from the project root:

```powershell
.\refresh-dev.ps1
```

The script restarts the local API, clears the API build output, serves `docs/` with browser caching disabled, and opens the scanner page.
